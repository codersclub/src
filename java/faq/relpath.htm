 Привет All,
 У меня проблема с сервлетами под Apache+JServ...
 В сервлете строчка
 FileInputStream xF = new FileInputStream("eee.frm");
 вызывает исключение
 java.io.FileNotFoundException: eee.frm (Системе не удается найти указанный файл)
 файл лежит в той же директории, что и сервлет.
 При выполнении данного сервлета под VisualCafe - все нормально!
 Какие нужны настройки для Apache и JServ, чтобы сервлет "находил" файл?
Ответ:
Вот сюда eee.frm и нужно было положить:
URL path = this.getClass().getClassLoader().getResource("");
System.out.println(path);
