Запуск внешних процессов в Java (d_k)
Пример кода при исполнении которого в ОС Windows открывается Блокнот.
class AppRunner
{
 public static void main(String args[])
 {
    Runtime r = Runtime.getRuntime();
    Process p = null;
    String cmd= "notepad";
    try
    {
      p = r.exec(cmd);
    }
    catch (Exception e){ e.printStackTrace(); }
  }
}
