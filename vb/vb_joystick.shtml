Пример демонстрирует использование API функций для Джойстика. Неплохая отправная точка для тех кто хочет сделать игрушку управляемую Джойстиком.
Функция joyGetDevCaps запускается при загрузке формы, чтобы опреледелить наличие Джойстика. Если функция выполнится успешно, то форма перересуется с параметрами Джойстика. Таймер используется для того, чтобы постоянно показывать текущую позицию.
Проект собран и откомпилирован на Visual Basic 5.0
