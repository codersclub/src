Огонь в реальном времени

Автор: Dertanyun

Программа рисует довольно реалистичный огонь с больним FPS (кадров в секунду). Работает в 16, 24, или 32-битном цветовом режиме. В примере используются API вызовы, которые большинство программистов, возможно, прежде не видели.

Компилятор: Visual Basic 5.0,  6.0.
