Как скрыть дочернее окно MDI

Для этого просто установите нулевые размеры MDI окна в его событии Form_Load:

form1.Move form1.Left, form1.Top, 0, 0

Для того, чтобы показывать и скрывать эту форму, воспользуйтесь элементом управления label:

If form1.Height = 0 Then
	form1.Move form1.Left, form1.Top, _
		form1!Label1.Width, form1!Label1.Height
Else
	form1.Move form1.Left, form1.Top, 0, 0
End If

Форма остаётся загруженной но нивидимой. Этот метод позволяет сделать всплывающее окно внутри MDI.
