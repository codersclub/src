SQL/Stored Proc/ADO - тестовый проект

Авторство:  LockwoodTech

Тестовый проект показывает высокопроизводительный доступ к данным для SQL Server используя команды ADO объектов и встроенные процедуры для запросов Insert, Update, Delete и Select. Представлены два метода для выборки - первый, использование recordset, другой - это использование только выходных параметров объекта, который работает намного быстрее, засчёт того, что отпадает необходимость создания объекта Recordset. SQL файл включён в архив исходника. Этот проект запускался на Northwind db.

Компилятор: Visual Basic 6.0.
