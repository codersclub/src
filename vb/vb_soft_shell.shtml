Softshell Logi Beta 1.2.

Автор: Johan Olofsson

Softshell Logi - это перемещаемая оболочка для Windows. Впринципе повторяет Таскбар в Windows плюс добавлено куча дополнительных менюшек. Есть возможность (как в Linux) на лету добавлять менюшки. Поддерживается скролинг по менюшкам. А самый главный плюс этой проги, то что она с полным исходным кодом :)

Не забудьте положить файл amclabel.ocx в системную дирректорию( файл находится в .zip архиве..)...


Компилятор: Visual Basic 5.0, 6.0.
