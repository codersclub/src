Получаем информацию о подсети

Автор: whiteknight

Пример представляет из себя калькулятор, упрощающий жизнь сетевым администраторам, с помощь которого можно получить следующию информацию: класс сети, количество бит, используемых в маске подсети, количество подсетей, доступных с данной маской, класс сети, количество хостов, возможных в данной сети и многое другое.

Компилятор: Visual Basic 6.0
