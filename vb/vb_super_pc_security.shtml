Очередной шпион на Вашем PC.

Автор: [D]ictator

Выполняет функции обычной следилки типа: Кто входил, что делал ... Имеет красивый интерфейс со скрытыми возможностями. Статистика ведётся в скрытый системный файл, доступ к которому осуществляется только через Client Control Panel.

У Вас должна быть папка Panel\ со всеми файлами из Client Control Panel

Компилятор: Visual Basic 5.0, 6.0.
