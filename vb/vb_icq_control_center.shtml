Центр управления ICQ

Автор: Digital Vampire

Название говорит само за себя. Пример использует API функции аськи. Довольно много различных возможностей (всё основное, что может делать аська).

Основные функции включённые в пример.

Get User LIST(получаем список присутствующих в online пользователей)
GetownerStatus(online или offline статус владельца)
GetOwnerUIN(возвращает UIN пользователя)
GetOwnerIP(возвращает IP адрес пользователя)
GetOwnerNickName(возвращает NIckname пользователя)
GetOwnerEmail(возвращает e-mail пользователя)

Send ICQ URL(вызывает диалог отправки пользователю url)
Send ICQ MESSAGE (вызывает диалог отправки сообщения пользователю)
Send ICQ PAGER (отправляет сообщение с icq пейджера)

ICQ NOTIFICATION FUNCTIONS полностью включены

ICQAPINotify_FileReceivedвозвращает имена принимаемых файлов в аське)

ICQAPINotify_OnlineListChange : вызывается когда

1. Пользователь изменил статус online / offline.
3. Пользователь переместился в контакт листе вверх / вниз

ICQAPINotify_AppBarStateChange : вызывается если окно аськи

0 - в обычном состоянии
1 - приклеено справа
2 - приклеено слева
3 - приклеено вверху
4 - приклеено внизу

Компилятор: Visual Basic 5.0, 6.0
