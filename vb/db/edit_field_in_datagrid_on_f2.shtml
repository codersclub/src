Как разрешить редактирование поля в DataGrid по F2

Обычно, для редактирования поля в DataGrid, достаточно кликнуть по нему. Однако, если вместо мышки пользователь использует клавиатуру, то желательно дать ему возможность редактировать поля через клавиатуру. Для этого достаточно добавить следующий код в событие grdDataGrid_KeyDown:

Private Sub grdDataGrid_KeyDown(KeyCode _
	As Integer, Shift As Integer)
	Select Case KeyCode
		Case vbKeyF2
			grdDataGrid.SelStart = 1
			SendKeys "{End}"
	End Select
End Sub
