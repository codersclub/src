Andromeda

Авторы: Ryan и Drew Lederman

Andromeda Remote File System это законченное Клиент/Серверное приложение. Andromeda использует Winsock для связи через протокол TCP/IP. Сервер поддерживает подключение сразу нескольких пользователей, и систему шифрования пароля. Возможности: Download/Upload файлов и директорий, перемещение файлов и директорий, переименование и удаление файлов и директорий, а также создание создание новых директорий, завершение/продолжение выполнения запущенных программ, и многое другое ... Так же можно запустить несколько серверов Andromeda. Поддерживается drag-and-drop между клиентом и сервером. Сервер может вести довольно подробный лог.

Эта программа чем-то напоминает CuteFTP, но не использует протокол File Transfer Protocol (FTP). Взамен этого мы встроили собственный протокол.

Компилятор: Visual Basic 6.0.

 

СКАЧАТЬ ИСХОДНИК(~400Kb)
