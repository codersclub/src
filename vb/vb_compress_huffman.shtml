Компрессия текстовых файлов методом Хаффмана

Автор: Fredrik Qvarfort

Вообще-то данным алгоритмом можно сжать файл любого формата, но на текстовых файлов этот метод показывает наибольшую производительность. Что самое интересное, производительность заключается в скорости сжатия. Вот например, приведённый ниже пример программы ужимает текстовый файл размером 1.6Мб за 1 секунду 8-0. .....и такое бывает :)

Компилятор: Visual Basic 5.0,  6.0.

 

СКАЧАТЬ ИСХОДНИК(~7Kb)
