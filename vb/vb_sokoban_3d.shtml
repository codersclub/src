Игра Sokoban 3D

Автор: Ерёмин Михаил

Высокоскоростной спрайтовый 3D движок встроенный в игру Сокобан. Эффект трехмерности достигается прорисовкой 2D объектов один на другой (получается вид как в StarCraft). Используется чистый VB и API (BitBlt), никаких C++ библиотек.

Компилятор: Visual Basic
