Цент управления Windows

Автор: Wazerface

Здесь проедставлена программа, которая является улучшенноой версией Wintasks. Добавлено много
новых возможностей и полностью изменён дизайн. Позволяет манипулировать с реестром, типа
разрешить/показать и запретить/скрыть для следующих разделов: Принтер и иконки панели управления,
установки windows таскбара, run и поиск в start menu, кнопка завершения сеанса и кнопка выключения
в start menu, документы, избранное, Internet Explore (IE) хистори, клик правой кнопкой на таскбаре,
и сохранение установок десктопа при выходе! Также, AGENT (Название этой программы) может
скрыть и показать иконку Сетевое окружение, и запретить или разрешить горячие клавиши windows.
Другие опции реестра включают: полная настройка свойств рабочего стола, типа позволять добавлять,
удалять и редактировать компоненты или нет, изменение обоев и т.д.! Также показывать или скрывать
некоторые страницы настроек, показывать или скрывать различные пункты в панели управления!
И это только то, что можно сделать через реестр! Эта программа также может соединять с интернетом,
посылать почту, открывать браузеры, устанавливать свойства интернета и открывать сайты! Панель
программы автоматически скрывается, если курсор мыши уходит за пределы окна программы. И если
опция 'Auto Hide' запрещена, AGENT будет сокращать десктоп, чтобы максимизированные приложения
не закрывали его!

Компилятор: Visual Basic 5.0, 6.0
