Как узнать, существует ли файл

Один из способов проверить существование файла, это открыть его на чтение, и проверить флаг ошибки. Если файл не существует, то будет сгенерирована ошибка.

Функция 'Dir$' выполняет туже самую задачу. 'Dir$' вернёт точное имя файла, если он существует, либо пустую строку, если такого файла нет. Например:

If Dir$("C:\WINDOWS\WIN.INI") _
                <> "" Then
                'file Win.ini exists!
         Else
                'file Win.ini does not exist!
         End If
