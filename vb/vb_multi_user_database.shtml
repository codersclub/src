Многопользовательская база данных

Автор:  Tim Feeley

Программа создаёт многопользовательскую базу данных, которую впоследствии может использовать Ваше приложение. Позволяет задавать пользователям различные параметры и при этом не требует наличия на компьютере какой-либо стандартной базы данных. В Архиве лежит описание (на английском) и исходный код проекта.

Компилятор: Visual Basic 6.0.
