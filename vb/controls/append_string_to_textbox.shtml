Как заставить Text Box прокручиваться вниз при добавлении в него строки

Следующий код используется для того, чтобы заставить TextBox автоматически прокручиваться вниз при добавлении в него нового кода:

''''Выбираем конец текста
MyTextBox.SelStart = Len(MyTextBox.Text)
''''Помещаем новый текст в конец
MyTextBox.SelText = NewText$
