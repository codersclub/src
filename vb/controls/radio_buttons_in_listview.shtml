Как использовать свойство радио кнопок в ListView

Небольшой кусок кода может заставить вести себя чекбоксы (checkboxes) в ListView наподобие радио кнопок (radio buttons). Установите в ListView свойства чекбосов в True и поместите следующий код в процедуру события ItemCheck:

Private Sub ListView1_ItemCheck(ByVal Item _
As MSComctlLib.ListItem)
Dim li As MSComctlLib.ListItem
For Each li In ListView1.ListItems
If li.Checked = True Then
If li <> Item Then li.Checked = False
End If
Next li
End Sub


В итоге галочка будет постоянно присутствовать только в одном чекбоксе списка.
