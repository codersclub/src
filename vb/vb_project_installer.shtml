Инсталяция для Вашего проекта

Автор: Sean Ferguson

Программа создаёт самораспаковывающуюся инсталяцию для Вашего Visual Basic проекта. Инсталяцию можно защищать паролем, который кодируется специальным ключом. Так же инсталятор может создавать ключи в реестре и и конки на рабочем столе и в меню "Пуск". Присутствует деинсталятор.

Компилятор: Visual Basic 6.0
