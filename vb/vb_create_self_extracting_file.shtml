Создание самораспаковывающегося файла.

Автор: Daniel Davies

Пример показывает, как создать исполняемый файл (.EXE), в который можно засунуть, например, содержимое дирректории. Программа не поддерживает никаких алгоритмов сжатия и архивации. Но для начала уже не плохо.

Компилятор: Visual Basic 5.0, 6.0.
