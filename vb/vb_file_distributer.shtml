Распределение файлов

Автор: Bill Jones

Данная программа создавалась для обновления на испытательных компьтерах в сети моего программного обеспечения. Работает как с промапленными дисками, так и с UNC. Основная задача - проверить версию программы на компьютерах в сети и при отличие версии скопировать туда обновление. Программа работает с файлом базы данных MS Access и использует API вызовы.

Компилятор: Visual Basic 6.0.
