Дополнительные сервисы на web серверах и сокеты в Java апплетах. 01.11.2000г.
Иногда возникает необходимость организации какого-нибудь дополнительного сервиса на Web сервере. При этом дополнительный сервер цепляется на какой-нибудь нестандартный порт и прослушивает его. В страничку, которую пользователь получает обычным образом, встраивается апплет, реализующий клиента для этого нестандартного сервера. Такую схему можно реализовать с помощью стандартного механизма сокетов языка Java
