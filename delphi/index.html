  Свои апплеты в панели управления 01.08.2002г.
Апплеты в панели управления, это обычные DLL, имеющие расширение .cpl (Control Panel Library) и располагающиеся в системной директории Windows. В свою очередь, в каждом файле cpl может храниться несколько апплетов. Cpl имеет единственную функцию точки входа CPlApplet(), через которую поступают все сообщения от панели управления.
Компилятор: Delphi 3.x

  Morph3D Screen Saver 30.05.2002г.
Morph3D Screen Saver - это скринсейвер для Windows. При запуске заставки на экране изображаются различные трехмерные вращающиеся объекты. Всего их семнадцать, в их список входят сферы, додекаэдр, икосаэдр, тор, спираль, куб, тетраэдр и другие. Все объекты состоят из 200 светящихся точек. Один объект периодически превращается в другой (эффект "morphing"). Программа написана с использованием функций WinApi, поэтому компилится под любой 32-битный Delphi.

  Как в Delphi изменить иконку у директории 18.05.2002г.
Обычно, для изменения вида папок в Проводнике используется файл desktop.ini. Сперва необходимо создать файл Desktop.ini и поместить в ту директорию, иконку которой мы хотим изменить. В программе для этого можно воспользоваться классом TIniFile и передать в него путь директории.

  Симфония на клавиатуре 01.04.2002г.
