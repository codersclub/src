Работаем с БД через библиотеки Microsoft ActiveX ADO.

Автор: Raymond Alexander

Пример включает в себя базу данных. Позволяет просматривать
структуру БД, просматривать и сохранять данные, восстанавливать и
записывать blob информацию в файл.Работает с Access 97/2000,
Oracle 8i (требует драйвер), и Microsoft SQL Server (Не тестировался).


Совместимость: Delphi 5.
