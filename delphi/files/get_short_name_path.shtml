Как определить короткое имя пути

Например:
c:\program files\common files\microsoft shared   в формате DOS будет выглядеть как
C:\PROGRA~1\COMMON~1\MICROS~1

Вот пример, как получить такое имя:

var
  Buffer: Array[0..MAX_PATH] of char;

GetShortPathName('c:\Program Files', @Buffer, SizeOf(Buffer));
ShowMessage('Short path name: ' + Buffer);
