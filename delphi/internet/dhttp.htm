Маленький многопоточный HTTP сервер

Copyright (c) 1999 Paul Toth (tothpaul@multimania.com) www.multimania.com/tothpaul

Простейший многопоточный HTTP сервер для Delphi32.
Поддерживает только метод GET.
Для работы с сокетами использует свой собственный модуль CrtSock.
Реализован в виде консольного приложения, выводит на экран протокол обработки запросов.
Совместимость: Delphi 2+
