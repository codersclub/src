Печать в сети

Если Вам известно имя принтера, то для печати в сети используется функция CopyFile либо пара функций CreateFile и WriteFile.

Печать при помощи CopyFile

  1. Вызываем CopyFile, указывая сетевое имя принтера, на котором будет выполняться печать и имя печатаемого файла.
  2. CopyFile имеет следующий синтаксис:
    CopyFile (szSrcFile, szUNCPrinterShare, FALSE);
  3. Файл для печати это szSrcFile. Сетевой принтер, это szUNCPrinterShare, который представляет из себя UNC имя, возвращаемое функцией WNetEnumResource.

Печать при помощи CreateFile и Writefile

  1. Для создания файла в сетевом принтере, вызываем CreateFile.
  2. Для записи печатаемых данных или документа во вновь созданный, вызываем функцию WriteFile.
  3. Закрываем файл в очереди заданий на печать.
