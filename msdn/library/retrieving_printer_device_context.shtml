Как получить контекст устройства принтера.

Когда пользователь выбирает пункт Печать (Print) в меню Файл (File), то Вы можете вызвать функцию PrintDlgEx, чтобы показать пользователю диалоговое окошко свойств печати. Это окошко позволит пользователю настроить параметры печати документа. Например, пользователь может указать диапазон страниц для печати, количество копий и т.д. А самое главное, что при помощи функции PrintDlgEx можно получить хэндл контекста устройства для выбранного принтера. Пример получения контекста устройства можно посмотреть в разделе "Использование диалогового окна свойств печати" в статье Использование стандартных диалогов.

Если Ваше приложение будет запущено в системе, которая не поддерживает PrintDlgEx, то воспользуйтесь функцией PrintDlg. Пример использования этой функции для получения контекста устройства можно посмотреть в разделе "Отображение диалога печати" в статье Использование стандартных диалогов.
