Функция CopyIcon.

Функция CopyIcon копирует указанную иконку из другого модуля в текущий.

HICON CopyIcon(      
    HICON hIcon
);

Параметры

hIcon
Дескриптор копируемой иконки.

Возвращаемое значение

В случае успеха, функция возвращает дескриптор скопированной иконки.

В случае ошибки, функция вернёт NULL. Для получения более подробной информации о ошибке, можно воспользоваться функцией GetLastError.


Замечания

Перед закрытием, приложение должно вызвать функцию DestroyIcon для освобождения различных системных ресурсов, связанных с иконкой.

Дополнительные сведенья

Заголовок Объявлена в Winuser.h, включена Windows.h
Библиотека User32.lib
Минимальные ОС Включена начиная с Windows 95, Windows NT 3.1
