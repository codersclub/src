Пример RAM диска для Windows 2000.

Пример является софтовым драйвером диска для Windows(r) 2000. Этот драйвер создаёт RAM диск нужногоразмера. Ramdisk можно использовать как любой другой, однако его содержимое сотрётся при перезагрузке компьютера или его выключении.

    Скачать Ramdisk.exe - 55 Кб

В архиве так же содержится уже откомпилированный драйвер, который можно установить следующим образом:

  1. В Control Panel, двойным щелчком запустите значёк Add/Remove Hardware.
  2. Выберите Add/Troubleshoot a device.
  3. Выберите Add a new device.
  4. Выберите No, I want to select the hardware from a list.
  5. Выберите Other devices, а затем кликните Next.
  6. Кликните Have Disk и укажите директорию, в которой находится файл Ramdisk.inf.

Система скопирует файл Ramdisk.sys в директорию %Systemroot%\System32\Drivers и загрузит драйвер.

Настройки драйвера производится через реестр. Так же все параметры можно отредактировать прямо в файле Ramdisk.inf. Далее представлены сами параметры с описанием:

BreakOnEntry REG_DWORD 0

    В данной версии драйвера не используется.

DebugLevel REG_DWORD 0

    В данной версии драйвера не используется.

DebugComp REG_DWORD 0xFFFFFFFF

    В данной версии драйвера не используется.

DiskSize REG_DWORD 0x100000 (1,048,576 1Mb)

    Объём диска в байтах.

DriveLetter REG_SZ Z:

    Буква диска.

RootDirEntries REG_DWORD 512

    Количество элементов в корневой директории.

SectorsPerCluster REG_DWORD 2

    Секторов в кластере.
