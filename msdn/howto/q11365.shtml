Как создать List Box без полосы прокрутки (Scroll Bar)

Чтобы создать list box без вертикальной полосы прокрутки, применяется операция NOT WS_VSCROLL. То есть из стандартного стиля создания списка вычитается стиль вертикального scroll bar, как показано ниже:

   (LBS_STANDARD & ~WS_VSCROLL)  // NOT WS_VSCROLL
