Valery Votintsev опубликован 04-02-2002 12:44 MSK            
Проще использовать не слои, а Document Object Model (почитай, заинтересуешься!)

Суть его - каждая фигулька на странице - это объект со своими свойствами и методами (прям как в Дельфи или VC++). И каждое свойство любой фигульки можно поменять. Например так:

document.MyObject.text.value = "хрю-хрю";

Valery Votintsev опубликован 06-02-2002 17:47 MSK            
<HTML>
<div id="test">This is English text</div>
<br>
<a href="#" onClick="document.all.test.innerText='This is English text'; return 1;">Set English</a>
<br>
<a href="#" onClick="document.all.test.innerText='Это Русский текст'; return 1;">Set Russian</a>
</HTML>
