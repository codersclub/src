Нулевые элементы считаются обычным For:

Procedure Zeros(N, M: Integer; var ZA, ZD: Integer);
Var i, j: Integer
Begin
ZA := 0;
ZD := 0;
For i := 1 to N do
For j := 1 to M do
Begin
If A[i,j] = 0 then Inc(ZA);
If D[j,i] = 0 then Inc(ZD)
End
End;

В результате ZA получает кол-во нулей в A[n,m] , а ZD - в D[m,n]
