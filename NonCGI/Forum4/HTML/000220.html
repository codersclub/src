А по поводу второго:

------------["Резать к чёртовой матери" тут (c)"Спасские ворота"]----------
Option Explicit

Private Type DataType
Arr(1 To 100) As Integer
Big As Long
Real As Double
Today As Date
S As String
End Type

Private Data As DataType

Private Sub Command1_Click()
Dim i As Integer
With Data
For i = LBound(.Arr) To UBound(.Arr)
.Arr(i) = i
Next i
.Big = &H12345678
.Real = 3.14159265358979
.Today = Now
.S = "Hello, world!"
End With
On Error GoTo Critical
Open "e:\data.dat" For Binary As #1
Put #1, 1, Data
Close #1
Exit Sub
Critical:
Call MsgBox("Обшибка записи!", vbCritical, "Облом")
End Sub

Разумеется, это происходит при нажатии на кнопку Command1...
Put #номер_файла, [номер_позиции], переменная - запись
Get --""-- - чтение
Номер позиции в файле отсчитывается не от нуля, а от единицы. Если не указан, то читается/записывается с текущей позиции.
Но это только для режима Binary. С Random там немного другая ситуация.
