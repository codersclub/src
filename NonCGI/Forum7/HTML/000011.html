Насколько помнится (сейчас под рукой Javы нет) примерно так:

import java.io.*;
........

try
{

// инициализация файла

FileOutputStream out =
new FileOutputStream("myfile.dat");
// записать данные
out.write('T');
out.write('e');
out.write('s');
out.write('t');

out.close(); // закрыть файл

}

catch(IOException e){}
