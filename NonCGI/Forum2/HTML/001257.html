Абсолютно аналогично.

Ну например :
есть
static UINT indicators[] =
{
ID_SEPARATOR,
ID_STATUS_1,
ID_STATUS_2,
ID_STATUS_3,
ID_STATUS_4
};

В BEGIN_MESSAGE_MAP(CMainFrame, CFrameWnd)пишешь
ON_UPDATE_COMMAND_UI_RANGE(ID_STATUS_1, ID_STATUS_4, OnUpdateStatusControls)

Дальше тело функции:
void CMainFrame::OnUpdateStatusControls(CCmdUI* pCmdUI)
{
switch(pCmdUI->m_nID) {
case ID_STATUS_1: pCmdUI->SetText(str1);
break;
case ID_STATUS_2: pCmdUI->SetText(str2);
break;
case ID_STATUS_3: pCmdUI->SetText(str3);
break;
case ID_STATUS_4; pCmdUI->SetText(str4);
break;
default: break;
}
}

Не забудь про afx_msg void OnUpdateStatusControls(CCmdUI* pCmdUI);
MainFrm.h
И имей в виду, что эти ID_STATUS_1 - ID_STATUS_4 должны быть описаны в resource.h строго друг за другом, иначе ничего не получится.
