Я как-то делал отправку sms через МТС, но это я делал на Perl.
Там похоже стоит проверка на сервер с которого пришла форма, т.е. надо с`имитровать
отправляющий сервер. Типа этого:

referer = 'http://www.mts.ru/sms/'
server = 'http://www.mts.ru'
content_encoding = 'gzip'

agent = 'Mozilla/4.0 (compatible; MSIE 5.5; Windows 98'
request = 'GET http://www.mts.ru/sms/sent.html?Posted=1&To=Телефон&Msg=TestFromMts&count=11&SMSHour=Час&SMSMinute=Минуты&SMSDay=День&SMSMonth=Месяц&SMSYear=Год'

На Perle у меня получилось. Ошибка такая же была.
