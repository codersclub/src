1. Обрадатывай LVN_ITEMCHANGED
2. Последовательность == кол-во строк * кол-во столбцов;

Пример : если bCheck == true, то можно чекать, иначе останется не чекнутой.
NM_LISTVIEW* pNMListView = (NM_LISTVIEW*)pNMHDR;
if(pNMListView->uOldState & 0x0000F000) {
if(ListView_GetCheckState(pMyList->GetSafeHwnd(), pNMListView->iItem)) {
if(!bCheck)
ListView_SetItemState(pMyList->GetSafeHwnd(), pNMListView->iItem, INDEXTOSTATEIMAGEMASK(1), LVIS_STATEIMAGEMASK);
}
}

Как работает селект при LVN_ITEMCHANGED
Лист - два столбца.
Был выделен второй итем(нуль-основан), выделяю пятый:
ЗАХОДЫ В ФУНКЦИЮ:
1. pNMListView->iItem == 1 (перв. столб.)
2. pNMListView->iItem == 1 (второй. столб.)
3. pNMListView->iItem == 4
