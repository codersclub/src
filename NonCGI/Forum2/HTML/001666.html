Например (VC++ 6.0)

Хочешь создать mas[5][10][15]; (храниться будут DWORDы)

LPDWORD lpDW = new DWORD[5* 10 * 15];
ZeroMemory(lpDW, sizeof(DWORD)*5*10*15);
lpDW + sizeof(DWORD)*(N) -> указ на N -ый элемент

Можно конечно и DWORD dw[5][10][15] без всякого new;
Но господин Страуструп не советует использовать многомерные массиви.

Alex2001 опубликован 02-11-2001 16:46 MSK            
Прототип:
<type> FunctionName(<type> Mas[D1][D2][D3]);
D1 - размерность 1 и т.д.

В main-не объявить массив:
<type> Mas[D1][D2][D3];

Вызов:
FunctionName(Mas);

В описании FunctionName:
<type> FunctionName(<type> Mas[D1][D2][D3])
{
<type> p;
...
p = Mas[1][4][4]; // Обратиться к элементу.
...
}
