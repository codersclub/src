migel опубликован 25-08-2001 18:39 MSK            
„ерез tmp. file:
1.create temp file (mktemp(); fopen();)
2.write your info (fwrite(const void* pBuff, size_t size, size_t count, FILE* pFile);)
3.open source file (FILE* fopen(const char* fName, const char* mode);)
4. copy src file to the tmp file (fread(...) fwrite());
5. close src file (fclose())
6. delete src file (unlink());
7. close tmp file (fclose());
8. rename tmp file (rename())
