Kosha опубликован 12-08-2001 02:37 MSK            
Из КЛАССА ты никак не передашь. Из объекта класса - масса способов. В зависимости от того, КАК надо передать и ОТКУДА...

Например:

class C2; //прототип

class C1
{
public:
...конструкторы-деструкторы..
void CallIt(C2 *Obj);
};

class C2{
...
void Function(...);
};

теперь, ты можешь вызвать CallIt с указателем на объект второго класса и уже из нее сделать Obj->Function(...);

Но это жутко криво...
