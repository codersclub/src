SUnteXx опубликован 29-10-2001 13:20 MSK            
The floor of 2.8 is 2.000000 // floor(double x);
The floor of -2.8 is -3.000000 // floor(double x);
The ceil of 2.8 is 3.000000 // ceil(double x);
The ceil of -2.8 is -2.000000 // ceil(double x);
The remainder of -10.00 / 3.00 is -1.000000 // double fmod( double x, double y );
The absolute value of -4 is 4 // double fabs( double x );
The absolute value of -41567 is 41567// double fabs( double x );
The absolute value of -3.141593 is 3.141593// double fabs( double x );

floor - Calculates the floor of a value
ceil - Calculates the ceiling of a value
fmod - Calculates the floating-point remainder
Calculates the absolute value of the floating-point argument
fabs - Calculates the absolute value of the floating-point argument
.
