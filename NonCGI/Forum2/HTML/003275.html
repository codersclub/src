Почему сообщения (MESSAGE) не доходят до View-класса?

Т.е. в "mainfrm.h" написано следующее:

...
//{{AFX_MSG(CMainFrame)
...
afx_msg DWORD OnGetCaretPosMessage(WPARAM wParam, LPARAM lParam); // Get caret pos
//}}AFX_MSG
...

в "mainfrm.cpp" написал следующее:
...
//{{AFX_MSG_MAP(CMainFrame)
ON_WM_CREATE()
ON_WM_CLOSE()
ON_COMMAND(ID_MY_TOOLBAR, OnMyToolbar)
ON_UPDATE_COMMAND_UI(ID_MY_TOOLBAR, OnUpdateMyToolbar)
ON_UPDATE_COMMAND_UI(ID_INDICATOR_POS, OnUpdatePosIndicator)
ON_UPDATE_COMMAND_UI(ID_INDICATOR_OVR, OnUpdateOvrIndicator)
ON_WM_SIZE()
ON_MESSAGE(WM_MY_MESSAGE1,OnMyMessage)
ON_MESSAGE(MSG_GETCARETPOS,OnGetCaretPosMessage)
ON_MESSAGE(WM_COPYDATA,OnReceiveData)
ON_WM_MOVE()
//}}AFX_MSG_MAP
END_MESSAGE_MAP()
...
...

DWORD CMainFrame::OnGetCaretPosMessage(WPARAM wParam, LPARAM lParam)
{
return MAKELONG(m_nPosLn,m_nPosCol);
}

Все работает замечательно. НО! Если я то же самое сделаю с классом CTextEditView.
То прb посылке приложению сообщения MSG_GETCARETPOS, ничего не происходит!!
Т.е. сообщение не доходит до этого класса.. Как тут быть??
