michl_m опубликован 19-02-2002 00:38 MSK            
Должно работать.

#pragma warning(disable:4786)
#include <string>
#include <vector>
using namespace std;
vector<string> sRegNum;

Обрати внимание на #pragma. Может быть, у тебя в проекте в Project Settings, на вкладке С/С++ General включено Warnings as errors. Поточу что warnings номер 4786 будет немеряно. Строчка #pragma... убивает этот warning.
