Не могу нормально создать окно.
Сделал мастером проект на базе диалога.
Наследовал CMyWnd:CWnd.

BOOL CTryApp::InitInstance()
{
CString wc;
wc = AfxRegisterWndClass(CS_HREDRAW|CS_VREDRAW);
CMyWnd* pWnd = new CMyWnd;
m_pMainWnd = pWnd;

pWnd->CreateEx(NULL,wc,"My window",
WS_VISIBLE|WS_CAPTION|WS_SYSMENU|WS_POPUPWINDOW,
CRect(0,0,200,200),NULL,NULL);

pWnd->ShowWindow(SW_SHOW);
pWnd->UpdateWindow();
return TRUE;
}

Ошибок никаких нет. Но вот фон окна не меняется, т.е. копируется рисунок, который был до создания окна. Что надо сделать, чтобы фон был нормальный. Предупредили, что нельзя вызывать CWnd::OnPaint().
