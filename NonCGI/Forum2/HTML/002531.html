zAg опубликован 30-12-2001 00:15 MSK            
по-моему это вообще бессмысленно:в этом случае sizeof(A) и sizeof(B) будут бесконечными - только представь: экземпляр класса A включает экземпляр класса B а тот в свою очередь опять класс A а тот снова класс B и так далее...
Так что здесь нужно использовать указатели:


--A.h--------
#ifndef fileAh
#define fileAh

class B;

class A{
B *m_b;
};

#endif //fileAh

аналогично :

------B.h------
#ifndef fileBh
#define fileBh

class A;

class B{
A *m_a;
};

#endif //fileBh

Только так и не иначе...
