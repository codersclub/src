Student опубликован 22-01-2002 17:39 MSK            
Товарищи !
Надо смотреть документацию по Intel. Там-же ЯСНО написано, что если после команды с использованием 32-битного регистра следующая команда использует часть этого-же регистра то образуется пауза в 8 тактов. Вот эта пауза и является главной задержкой при работе с bool.

>>
[BOOL]
call ....
test EAX, EAX
; >> паузы нет
jnz....

[bool]
call ...
and eax, 0xFF ;<<
test al, al
; >> ПАУЗА 8 ТАКТОВ
jnz ...
