Kostik опубликован 19-11-2001 05:23 MSK            
DWORD dwStart = 0;
DWORD dwRes = 0;
При старте потока делаешь DWORD dwStart = GetTickCount();

При выходе из потока
dwRes = (GetTickCount() - dwStart) / 1000; сек.
