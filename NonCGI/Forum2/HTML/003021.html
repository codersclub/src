Координаты кнопки клос пытался получить следующим образом:
RECT R;
GetWindowRect(hWnd, &R);
R.right -= R.left; // Длинна окна
R.bottom -= R.top; // Высота окна
R.left = 0; // н0ль (клиентские ж координаты:)
R.top = 0; // н0ль (клиентские ж координаты:)
.
.
.
R.right -= 2; // 2 - отступ от рамки окна
R.top += 2; // тоже самое
R.left = R.right - GetSystemMetrics(SM_CXSMSIZE); // GetSystemMetrics(SM_CXSMSIZE) - размеры кнопки
R.bottom = R.top + GetSystemMetrics(SM_CYSMSIZE); // GetSystemMetrics(SM_CYSMSIZE) - размеры кнопки

Но, почему-то, координаты не точны, т.е. не точны размеры кнопки. Может надо определять размеры не через SM_CXSMSIZE? Пробовал SM_CXSIZE, не подходит, ...
