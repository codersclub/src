Flex Ferrum опубликован 22-10-2001 11:42 MSK            
В принципе можно. Только с одним условием - второй параметр должен быть константным. То есть, скорее прокатит такая фишка:

template <class T, int size> class MyTemplate
{
T a;
public:
MyTemplate() : a(size) {;}
};

А ты сделай так:
template<class T, typename IT constant>
...

К томуже, если ты в любом случае имеешь в виду скалярные типы (int, double и т. п.), то никаких проблем быть не должно.
