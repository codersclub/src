Как программно сделать скролинг у формы.

Компилятор: C++ Builder

Чтобы заставить скроллбары формы реагировать на стандартные клавиши PageUp, PageDown, Home, End, Up, и Down, необходимо использовать сообщение WM_VSCROLL в событии формы OnKeyDown:

void __fastcall TForm1::FormKeyDown( ... )
{
   if (Key == VK_KEYDOWN)
      SendMessage(Handle, WM_VSCROLL, SB_LINEDOWN, NULL);
   if (Key == VK_NEXT)
      SendMessage(Handle, WM_VSCROLL, SB_PAGEDOWN, NULL);
   // etc
}
