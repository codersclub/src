Как сделать, чтобы по нажатию Enter активизировался следующий компонент (как и по таб)?

Автор: OlegGG
Исходная ссылка: - - -

Ставим свойство формы KeyPreview в true.
А в OnKeyUp формы пишем:
if (Key==VK_RETURN)
        {
        Form1->Perform(WM_NEXTDLGCTL, 0, 0);
        }
