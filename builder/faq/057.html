Как использовать свои курсоры?

Автор: OlegGG
Исходная ссылка: - - -

Пример (здесь "mycur.ani" имя файла курсора):
HANDLE cur = LoadImage(0, "mycur.ani",IMAGE_CURSOR,0,0,LR_DEFAULTSIZE + LR_LOADFROMFILE);
if (cur)
        {
        Screen->Cursors[1] = cur;
        Form1->Cursor = 1;
        }
else
        {
        ShowMessage("oblom");
        }
