Как считать заголовок окна по его хэндлу?

Автор: KAV
Исходная ссылка: - - -

Пусть у нас есть OurHandle - хэндл окна, заголовок которого нужно считать. Запишем этот заголовок в Label1:
char tt[30];  // переменная-буфер для хранения текста заголовка
GetWindowText(OurHandle,tt,30);  // параметры: хэндл, буфер, сколько символов максимум
Label1->Caption = tt;  // и присваиваем этот текст тексту компонента Label1
