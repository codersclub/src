Как программно очистить корзину?

Автор: OlegGG
Исходная ссылка: - - -

В shell32.dll есть функция SHEmptyRecycleBin.
Пример:

#include <Shellapi.h>

if(SUCCEEDED(SHEmptyRecycleBin(NULL,NULL,SHERB_NOCONFIRMATION + SHERB_NOPROGRESSUI + SHERB_NOSOUND))
        {
        ShowMessage("OK");
        }
else
        {
        ShowMessage("ERR");
        }

Первый аргумент функции - HWND окна, которому будут принадлежать все диалоги, вызываемые функцией (может быть равен NULL).
Второй - диск на котором будет очищена корзина (может быть равен NULL).
Третий - флаги:
SHERB_NOCONFIRMATION - не подтверждать,
SHERB_NOPROGRESSUI - не показывать окошко с прогрессбаром,
SHERB_NOSOUND - не уведомлять звуком об окончании операции.
