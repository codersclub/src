Как определить координаты курсора мышки.

Компилятор: C++ Builder

Для этого существует API функция GetCursorPos. В функцию необходимо передать указатель на структуру POINT (в BCB, TPoint это тоже самое, что и в windows структура POINT). Ниже приведён пример, для использования которого надо поместить на форму TLabel и TTimer, а в обработчик события OnTimer поместить следующий код. При каждом движении мышки, будут отображаться её координаты.

    void __fastcall TForm1::Timer1Timer(TObject *Sender)
    {
      TPoint pt;
      GetCursorPos(&pt);

      Label1->Caption = "(" +
                        IntToStr(pt.x) +
                        ")(" +
                        IntToStr(pt.y) +
                        ")";
    }
