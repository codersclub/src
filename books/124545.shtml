В книге даются исчерпывающие справочные сведения по версии языка Object Pascal, используемой в Delphi 5. Попутно приводятся данные по 16-разрядной версии Object Pascal в Delphi 1, не потерявшей значения при разработке прикладных программ для маломощных компьютеров.
Излагаются сведения по синтаксису языка, операциям и операторам. Даются сведения по всем типам Object Pascal. Подробно рассматривается работа с исключениями, с текстовыми и двоичными файлами, со строками разных типов, массивами (в том числе открытыми и динамическими), множествами, записями, классами. Обсуждается обработка и генерация сообщении Windows. Изложение сопровождается многими примерами. Рассматривается более 200 функций языка Object Pascal, библиотечных функций Delphi и функций API Windows. Как справочник книга полезна пользователям любой квалификации: от начинающих до опытных разработчиков.
