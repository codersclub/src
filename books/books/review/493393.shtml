Книга является систематизированным справочником по всем элементам языка программирования Visual Basic (VB), входящим, как в автономную версию VB, так и в основные реализации Visual Basic for Applications.
Помимо тех официальных сведений, которые можно найти в документации и в справочной системе, книга содержит конкретные примеры, рекомендации и маленькие хитрости программиста, а также поправки к справочной системе, которая, увы, не без ошибок. Книга предназначена для всех специалистов, занимающихся разработкой приложений с применением Visual Basic и Visual Basic for Applications.
