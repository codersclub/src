В книге дано детальное описание двух языков программирования для сетевых приложений. Языки Java и JavaScript служат для создания программ-агентов (апплетов), способных свободно перемещаться по компьютерным сетям.
Изложение ведется методом сравнения особенностей этих языков, но в то же время каждому из них посвящена отдельная часть книги. Основное внимание уделяется разработке апплетов, необходимых любому разработчику Web-страниц. В руководстве приведено множество детально рассмотренных фрагментов программ, которые можно применять без изменения в пользовательских апплетах.
