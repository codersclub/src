содержание
Автор шаг за шагом показывает использование JDBC для интеграции промышленных баз данных с Java, SQL и другими передовыми Web-технологиями.
Подробное руководство с большим числом примеров JDBC - связность баз данных Java - предлагает профессионалам полное ноу-хау JDBC: от основных соединений, инструкций и результатов до таких современных технологий, как метаданные, многоуровневые архитекторы и RMI.
