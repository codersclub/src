Visual Basic 6 - последняя версия одного из наиболее популярных во всем мире языков программирования - предоставляет в распоряжение программиста инструментарий для разработки визуальных интерфейсов доступа к абсолютно любым данным.
В Visulal Basic 6 включены недавно появившиеся объекты управления данными ActiveX (ActiveX Data Objects - ADO) для доступа к базам данных из программы. В книге рассказывается, как перенести имеющиеся программы для работы с базами данных на новую платформу, а также о новых методах, которые помогают с помощью Visual Basic обеспечить единообразный доступ из вашей программы к самым разным источникам информации. Книга проведет вас от базового уровня знаний до вершин мастерства. Прочитав ее, вы сможете нетривиально использовать некоторые из новейших технологий и научитесь работать с данными, находящимися в различных электронных хранилищах. Таким образом, на первом этапе вы ознакомитесь с хорошо зарекомендовавшей себя технологией доступа к данным Data Access Objects (DAO), а только после этого речь пойдет о новой и стратегически важной технологии ActiveX Data Objects (ADO). Настоящее издание адресовано тем, кто собирается создавать с помощью Visual Basic 6 программы для работы с базами данных, а также тем, кто:
  • хочет научиться использовать Visual Basic для доступа к базам данных;
  • имеет опыт программирования, но пока не работал с Visual Basic;
  • работал с предыдущими версиями Visual Basic и хочет узнать, в каких направлениях развивается Visual Basic 6 в рамках парадигмы универсального доступа к данным (Universal Data Access - UDA).
