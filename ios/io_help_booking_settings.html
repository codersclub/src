Booking Configuration



Fig. 1
Main menu for booking configuration:
- Resource Group list(Fig.2) - for listing and viewing/adding/updating resource groups in DB
- Resources list(Fig.4) - for listing and viewing/adding/updating resources in DB
- Event groups list(Fig.6) - for listing and viewing/adding/updating event groups in DB
- Events list(Fig.8) - for listing and viewing/adding/updating events in DB
- New Resource Group(Fig.3) - quick link to adding new resource group to DB
- New Resource(Fig.5) - quick link to adding new resource to DB
- New Event Group(Fig.7) - quick link to adding new event group to DB
- New Event(Fig.9) - quick link to adding new event to DB



Fig. 2
List of resource groups. You can click to to enter in edit mode. For adding new resources group you must click to "Add new resources group" link, locates in the bottom of list



Fig. 3
Adding new resources group or updating existing one. If you want return to list of resources groups without save changes you can click to "Return to resources groups list".


Fig. 4
List of resources. You can click to to enter in edit mode. For adding new resource you must click to "Add new resource" link, locates in the bottom of list



Fig. 5
Adding new resource or updating existing one. If you want return to list of resources without save changes you can click to "Return to resources list".


Fig. 6
List of events groups. You can click to to enter in edit mode. For adding new events group you must click to "Add new events group" link, locates in the bottom of list



Fig. 7
Adding new events group or updating existing one. If you want return to list of events groups without save changes you can click to "Return to events groups list".


Fig. 8
List of events. You can click to to enter in edit mode. For adding new event you must click to "Add new event" link, locates in the bottom of list



Fig. 9
Adding new event or updating existing one. If you want return to list of events without save changes you can click to "Return to events list".
