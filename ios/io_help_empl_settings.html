Employees Configuration



Fig. 1
Main menu for employees configuration:
- New Employeer - for adding new person to DB (Fig.2)
- Employees list(Fig.3) - for listing and viewing/adding/updating persons in DB
- Departments list(Fig.4) - for listing and viewing/adding/updating departments in DB
- Employees search - for searching persons (not purposed in this version)
- Employees status list(Fig.6) - for listing and viewing/adding/updating status in DB
- Locations list(Fig.8) - for listing and viewing/adding/updating locations in DB



Fig. 2
Adding new person to DB. Perhaps firstly need to add:
1)Locations - Locations List(Fig.1) -> Add new location(Fig.8)
2)Departments - Departments List(Fig.1) -> Add new department(Fig.4)
3)Status - Employees Status List(Fig.1) -> Add new status(Fig.6)



Fig. 3
List of persons. You can click to to enter in edit mode. For adding new person you must click to "Add new employeer" link, locates in the bottom of list



Fig. 4
List of departments. You can click to to enter in edit mode. For adding new department you must click to "Add new department" link, locates in the bottom of list



Fig. 5
Adding new department or updating existing one. If you want return to list departments without save changes you can click to "Return to departments list".


Fig. 6
List of employees statuses. You can click to to enter in edit mode. For adding new status you must click to "Add new status" link, locates in the bottom of list



Fig. 7
Adding new status or updating existing one. If you want return to list statuses without save changes you can click to "Return to status list".


Fig. 8
List of locations. You can click to to enter in edit mode. For adding new location you must click to "Add new location" link, locates in the bottom of list



Fig. 9
Adding new location or updating existing one. If you want return to list locations without save changes you can click to "Return to locations list".
