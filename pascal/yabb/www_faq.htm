PHP


Почему недоступны параметры, переданные PHP скрипту? В классическом PHP любая переменная, переданая странице через GET / POST / Cookie, немедленно регистрировалась и была доступна на странице. Между тем, в новых версиях PHP и на нестандартных конфигурациях, эта возможность обычно выключена, что приводит к частым проблемам.
За включение этой возможности отвечает параметр register_globals в файле php.ini, который, в свою очередь, находится в папке, в которой установлен Windows. Если у Вас есть доступ к этому файлу, достаточно в нем установить register_globals=1.
Если же у вас нет прав доступа к этому файлу (если сайт находится на сервере хостера например), вам придется использовать массивы _GET и _POST для получения нужных значений.

Пример:
File.html
<html>
<head>
<title>Example</title>
</head>
<body>
<form action="file.php" method="post">
Ваше имя:<br>
<input type=text name="name" maxlength=20 size=30><br>
<input type=submit value="Продолжить">
</form>
</body>
</html>

File.php
<html>
<head>
<title>Example</title>
</head>
<body>Hello
<?
$name = $_POST["name"];
echo $name;
?>!
</body>
</html>

Как получить файл от клиента, используя PHP? Для начала, пусть у нас будет форма в файле upload.html:
<html><body>
<form enctype="multipart/form-data" action="upload.php" method=post>
<input type="hidden" name="MAX_FILE_SIZE" value="1000">
File Name: <input type="file" name="userfile">
<input type="submit" value="Send">
</form>
</body></html>

Эта форма будет посылать выбранный файл нашему скрипту.
Когда PHP получает такой файл, он записывает принятый файл во временный, и устанавливает несколько переменных: (в нашем случае)
$userfile_size - Размер файла в байтах
$userfile_name - Имя файла на компе юзера (c:\file.txt например)
$userfile - Имя временной копии файла на сервере. Её можно открыть как обычный файл.

Вот пример скрипта, выводящего принятый файл:
echo " <br> Size: $userfile_size<br>
Old Name: $userfile_name<br>
Temp Name: $userfile<br><pre>";
echo implode ("",file($userfile)); // Выводит файл

Коротко о возможных подводных камнях:
Текст будет аплоадится всегда, а вот при аплоаде бинарных файлов есть шанс получить файлы "битыми". Происходит это потому, что Русский Апач перекодирует все подрят. Например 0x00 он заменит 0x20 (пробел) Чтобы с этим бороться, надо в файле настроек httpd.conf прописать что-то типа:
<Location />
CharsetRecodeMultipartForms Off
</Location>

Как в PHP работать со случайными числами? Для этого есть функции rand() srand(int) и getrandmax()

Для начала, генератор случайных чисел нужно проинициализировать с помощью srand (сколько-то). Для того чтобы убедится, что числа будут действительно случайными, можно например, в начале скрипта вызвать srand(date("s"))

После этого функция rand() будет генерировать случайные числа в пределах 0<->getrandmax() Для того, чтобы изменить максимальный предел случайных чисел, достаточно взять остаток деления получившегося числа на новый предел. Например, rand() % 100 даст случайный номер от 0 до 100.

JavaScript


Как работать с файлами с помощью JavaScript? Ответ чрезвычайно прост – никак!. Из соображений безопасности в клиентской версии JavaScript, функции по работе с файлами просто отсутвуют.
Между тем, если сторонний файл имеет расширение .js, его можно включить в скрипт директивой include. В этом случае файл будет загружен и выполнен как часть скрипта.

From: javascript - как прочитать внешний текстовый файл

Как узнать координаты курсора мыши в JavaScript?
<html>
<head>
<script>
function mice() {
var x = document.body.scrollLeft + event.x;
var y = document.body.scrollTop + event.y;
alert('x:' + x + ' y:' + y);
}
</script>
</head>
<body>
<a href="" onmouseover="mice();">Link</a>
</body>
</html>
При наведении курсора на ссылку выскочит окошко с координатами курсора.

Как открыть две ссылки в разных фреймах? frames.htm
<html>
<head>
<title>Frames</title>
</head>
<frameset cols="300,*">
<frame src="menu.htm" name="fr1">
<frame src="body.htm" name="fr2">
</frameset>
</html>

menu.htm
<html>
<head>
<script language="JavaScript">
function loadtwo(page1, page2) {
parent.fr1.location.href=page1;
parent.fr2.location.href=page2;
}
</script>
</head>
<body>
<input type="button" value="Click me!" onClick="loadtwo('newmenu.htm','newcell.htm')">
</body>
</html>
При нажатии на кнопку в двух фреймах будут открыты страницы newmenu.htm и newcell.htm

Как установить иконку для сайта? Обычно бывает достаточно создать в одной директории с сайтом файл favicon.ico, который и будет нужной иконкой.
В случае, если это не возможно, нужно прописать адресс иконки в хедерах страницы следующим образом:
<Link Rel="Shortcut Icon" HREF="http://somewhere.com/myicon.ico">

From: Список вопросов...

Как работать с номерами ICQ? Вставить в свою страничку цветочек ICQ индифицирующий находится ли человек он-лайн, можно командой
<img src="http://web.icq.com/whitepages/online?icq=номер_icq&img=5" border="0">.
Тут номер_icq обозначает номер нужного человека, а 5 - стиль иконки. В данном случае - цветочик.
Для того чтобы увидеть дополнительную информацию о выбраном человеке, на иконку можно поставить ссылку, следуйщим образом:
<a href="http://wwp.icq.com/wwp/1,,cluin,00.html?Uin=номер_icq"></a>.

Установка Web-сервера


Как установить и настроить IIS? Перед установкой IIS / PWS прежде всего надо помнить что не все версии Windows поддерживают такую установку.
Так, если верить Microsoft, системы, работающие под Windows Me и Windows XP Home Edition, не могут стать сервером.
Установку IIS / PWS можно найти на диске Windows в разделе "Установка дополнительных компонентов" или в папке /add-ons/pws/.
После установки адрес http://127.0.0.1/ (localhost) будет ассоциирован с папкой /Interpub/wwwroot/.
Также, сервер будет распознавать файлы .asp как сценарии на языке ASP. Настройки PWS будут доступны при нажатии на соответсвующую иконку в трее, в то время как настройки IIS можно найти в панели управления. Более подробную информацию можно найти на сайте Microsoft; в электронной библиотеке MSDN и на других сайтах в интернете.
