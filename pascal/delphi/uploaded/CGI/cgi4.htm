WWW-Authenticate

Alors, le browser reзoit une demande d'authentification : WWW-Authenticate: Basic realm="/MyRealm"
En rйponse, le CGI reзoit le profile/mot de passe dans la variable d'environnement HTTP_AUTHORIZATION 
  HTTP_AUTHORIZATION=Basic dXNlcjpwYXNzd29yZA==
Pour valider cette information, il faut la dйcoder...elle est en effet au format Base64.

Petit rappel sur les bases (cours de mathйmatique de primaire ! ):
  • Habituellement on travaille en base 10 : 0..9
  • En informatique on utilise couramment la base 16 : 0..9,'A'..F'
  • Sur internet on utilise la base 64 : 'A'..'Z','a'..z','+','/'
Vous trouverez dans le programme exemple une unitй Base64 qui prend en charge de codage/dйcodage d'un type String en base 64.

le texte ci-dessus une fois dйcodй donne tout simplement : "user:password"

Il ne reste plus au CGI qu'a accepter le mot de passe en envoyant les informations demandйes ou renvoyer l'erreur 401.
