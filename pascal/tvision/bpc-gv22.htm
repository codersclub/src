	 Graphics Vision V2.2  for  Turbo/Borland Pascal 7.0
	-----------------------------------------------------

Graphics Vision is a Turbo Vision 2.0 compatible library that works
in graphics mode, with lots of advanced features and improvements.

I've used parts of the following sources when creating Graphics Vision:
Borland Pascal 7.0 Runtime Library Source Code (I didn't have BP 7.01
when I started coding Graphics Vision), Super Vision for Turbo Pascal 6.0
and modified Turbo Vision 2.0 by ACD Group. However, most of the sources
included here are coded completely by me.
