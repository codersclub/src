Как работать с файлами с помощью JavaScript? (Torio)
Ответ чрезвычайно прост – никак!
Из соображений безопасности в клиентской версии JavaScript, функции по работе с файлами просто отсутвуют.
Между тем, если сторонний файл имеет расширение .js, его можно включить в скрипт директивой include.
В этом случае файл будет загружен и выполнен как часть скрипта.
