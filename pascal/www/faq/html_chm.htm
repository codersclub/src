Как создать chm файл? (Torio)
Для создания chm файлов можно воспользоваться программой HTML Help Workshop,
входящей в установку Windows 2000/XP.

Если у вас нет этой программы, ее можно скачать с сайта Microsoft:
htmlhelp.exe - 3.5Mb
