Как узнать координаты курсора мыши в JavaScript? (Torio)
Для этого можно воспользоваться объектом event:
<html> 
<head> 
<script> 
function mice() { 
var x = document.body.scrollLeft + event.x; 
var y = document.body.scrollTop + event.y; 
alert('x:' + x + ' y:' + y); 
} 
</script> 
</head>
<body>
<a href="" onmouseover="mice();">Link</a> 
</body> 
</html>
При наведении курсора на ссылку всплывет окошко с координатами курсора.
