Какие есть бесплатные хостинги? (Torio)
Прежде всего мало кто не слышал о narod.ru. Относительно средний хостинг:
Нет CGI, скорость маленькая, при закачке файла вылетает страница с баннером.
С другой стороны, хостинг дает 100mb доступного пространства и имя типа yourname.narod.ru.

By.ru - неограниченое место, однако, с ограничением размера файл до 250kb. Имя типа yourname.by.ru, бесплатные доски объявлений, чаты и т.д.

Chat.ru - 10mb под сайт, и столько-же под почту. Основным плюсом является отсутствие рекламы.

Fatal.ru - 30mb доступного пространства. Неплохая скорость, поддержка SSI, CGI. Имя типа yourname.fatal.ru. Есть почта.

Есть и другие, менее популярные хостинги. На мой взгляд, для изучения CGI наиболее подходит домашний сервер, а для создания серьезного ресура - платный хостинг.
Как говорят, бесплатный сыр бывает только в мышеловке ;)
