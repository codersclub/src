Перевод процессора в защищённый режим

Автор: Петров С.В.

ASM-программа переводит процессор в защищённый режим, начинает выполнение задачи кольца 0, по нажатию переключает между собой задачи: "1"-задача 3-го уровня, "2"-задача 3-го уровня,"0"-задача 0 уровня, V-задача виртуального 8086 с прерыванием int 10h (вывод строки), выход по F10. Задачи разнесены по LDT, в программе перепрограммируется контроллер прерываний, часто повторяющиеся процедуры вынесены в debug.mac. Программа написана по книге Рудакова Финогенова "Программируем на языке ASSEMBLER IBM PC" 3-е издание. У кого возникнут вопросы-пишите, по возможности отвечу.
