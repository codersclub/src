Часто задаваемые вопросы и ответы по программированию на VB
Последнее обновление: 20.05.03

Split

Split (Строка, разделитель)
Возвращает массив

Chr
Chr (Код символа)
Возвращает строку, содержащую символ.

Instr
Instr ([Старт,] Строка1, Строка2 [, 0/1] )
Возвращяет номер позиции начала Строки2 в составе Строки1, или 0, если таковая не обнаружена.

Left
Left (Строка, Количество символов)
Возвращает строку, извлеченную из строки-параметра.

Len
Len (Строка/Имя переменной)
Возвращает количество символов или байт в строке.

LTrim
LTrim (Строка)
Удаляет лидирующие пробелы из строки-параметра.

Mid
Mid (Строка, Старт [, Длинна])
Возвращает строку, извлеченную из строки-параметра, начиная с символа в позиции Старт, и содержащую Длинна символов.

Right
Left (Строка, Количество символов)
Возвращает строку, извлеченную из строки-параметра.

RTrim
RTrim (Строка)
Удаляет замыкающие пробелы из строки-параметра.

Str
Str (Число)
Преобразует числовое значение в строку.

StrComp
StrComp (Строка1, Строка2 [,Сравнение])
Сравнивает строки и возвращает число-результат. Режим сравнения зависит от параметра Сравнение.

StrConv
StrConv (Строка, Конвертирование)
Преобразует строку. Вид преобразования зависит от параметра Конвертирование.

String
String (Число, Символ)
Создает строку из заданного числа заданных символов.

Trim
Trim (Строка)
Удаляет лидирующие и замыкающие пробелы из строки-параметра.
