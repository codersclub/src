Q:Как удалить файл?
A1 (=Alex=):

Private Declare Function DeleteFile Lib "kernel32" Alias "DeleteFileA" (ByVal lpFileName As String) As Long

Private Sub Command1_Click()
DeleteFile "C:\1\1\conv.htm"
End Sub

A2 (Crew):
' Путь к файлу может содержать символы "*" и "?" для удаления файлов по маске.
Private Sub Command1_Click()
Kill "C:\1\1\conv.htm"
End Sub
