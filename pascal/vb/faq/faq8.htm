Q: Как сделать форму, которую нельзя закрыть?

A:

Если установить свойство формы ControlBox в False, то кнопки минимизации и максимизации
так же станут невидимыми. Однако, иногда бывает необходимо оставить возможность максимизировать и
минимизировать форму, но при этом не разрешать её закрывать. Для этого просто добавьте следующий
код в событие Query_Unload:

' в VB3 следующую строчку следует раскоментировать
' Const vbFormControlMenu = 0
Private Sub Form_QueryUnload(Cancel As Integer, UnloadMode As Integer)
If UnloadMode = vbFormControl_Menu Then
Cancel = True
End If
End Sub
