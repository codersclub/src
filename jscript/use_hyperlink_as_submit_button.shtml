Как использовать ссылку в качестве кнопки отправки формы

Если Ваша страничка разработана с использованием нестандартных элементов управления, то возможно обычная кнопка может не подойти. Следующий пример показывает, как в качестве такой кнопки использовать ссылку:

<HTML>
<BODY>
<FORM name=frmMyForm method=post action="newpage.asp">
<input type=hidden name=hiddenfield value="hiddenvalue">
</FORM>
<a href="JavaScript:document.frmMyForm.submit();">
Go to other page</a>
</BODY>
</HTML>
