Один из способов предварительной загрузки картинок

Каритнки можно предварительно загрузить создав новый объект image и указывать его в качестве источника. Чтобы обеспечить проверку на отключённость JavaScript, необходимо добавить условие, как показано ниже:

<script>
if (document.images) {
            welcome = new Image();      
            welcome.src = "images/welcome.gif"; 

            welcome2 = new Image();      
            welcome2.src = "images/welcome2.gif";
 }
</script>
