Как по клику менять картинку

Всё, что нам понадобится, это два GIF-а и небольшой скрипт. Следующий код изначально показывает логотип Microsoft. Если кликнуть на картинку, то сработает событие onclick и будет вызвана функция swapImage(). Функция определяет, какая картинка должна быть показана.

<HTML>
<HEAD>
<SCRIPT LANGUAGE=JavaScript>
intImage = 2;
function swapImage() {
switch (intImage) {
 case 1:
   IMG1.src = "http://www.microsoft.com/library/toolbar/images/mslogo.gif";
   intImage = 2;
   return(false);
case 2:
   IMG1.src = "http://msdn.microsoft.com/msdn-online/start/images/msdn-logo.gif";
   intImage = 1;
   return(false);
 }
}
</SCRIPT>
</HEAD>
<BODY>
<IMG id="IMG1" name="IMG1" src="http://www.microsoft.com/library/toolbar/images/mslogo.gif" 
 onclick="swapImage();">
</BODY>
</HTML>
