Конструируем собственное окно.

Следующий пример, позволяет создавать окна, отличные от стандартных окон браузера, чтобы посетитель странички мог лучше сосредоточиться на содержимом сайта. При этом сохраняются все возможности стандартных окон.

Так же Вам потребуется скачать этот zip-файл, который содержит дополнительный код и картинки для заголовка окна.

Настройка:

theURL URL страницы, которая будет загружена в окне
wname внутреннее имя окна
W размер окна по горизонтали
H размер окна по вертикали
windowCERRARa Три GIF-а, отражающие три состояния кнопки закрытия окна.
windowCERRARd
windowCERRARo
windowBORDERCOLOR Цвет края окна по умолчанию
windowBORDERCOLORsel Цвет края при перемещении окна
windowTITBGCOLOR Цвет заголовка окна
windowTITBGCOLORsel Цвет заголовка при перемещении окна
