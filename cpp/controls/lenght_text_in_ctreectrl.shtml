Ограничение длины вводимого текста

Для ограничения вводимого текста, Вы должны перехватить TVN_BEGINLABELEDIT сообщение. В его обработчике вызвать функцию LimitText(), которая принадлежит окну редактирования. 


void COutline::OnBeginLabelEdit(NMHDR* pNMHDR, LRESULT* pResult) 
{
 TV_DISPINFO* pTVDispInfo = (TV_DISPINFO*)pNMHDR;
 // Устанавливаем лимит в 127 символов
  GetEditControl()->LimitText(127);

 *pResult = 0;
}
