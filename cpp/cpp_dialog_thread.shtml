Простой пример рабочего потока

Автор: Rick Schneewind.

Компилятор: Visual C++ 6.0

Пример демонстрирует, как запустить рабочий поток из диалога, при этом сохраняя работоспособность диалога, то есть работая в фоновом режиме.

Первый из трёх циклов, запускаемый по событию OnButton, путём вызова функции из CloopClass. Он демонстрирует, как при работе данного цикла невозможно перемещать диалог. Второй и третий циклы демонстрируют, как при их работе диалог можно свободно перемещать.
