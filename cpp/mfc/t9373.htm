At 08:05 2/2/97 +0200, Amir Shoval wrote:
>Environment: VC4.2b ,Windows 95

>Where can I find information about the changes in comctl32.dll and when
>will these changes will be reflected in MFC?

The beta version of COMCTL32.DLL that you have is documented in a file called
CC_INFO.DOC, which is in the ActiveX 3.01 SDK.  The documentation is
a preliminary prerelease, and is bound to change--just like the control
implementations.

MFC will support the new controls in the first release of MFC after the 
controls are finally released.  MFC isn't intended to support beta
operating system components: doing so results in huge headaches for everyone,
as evidenced by the 4.2b patch.
