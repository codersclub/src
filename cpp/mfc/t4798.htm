I am using TrimRight in one of 16-bit applications. The function works 
fine. But you must make sure installing the MFC250.DLL made at 4/26/95 
into windows directory. Specially in Win95, an older version of 
MFC250.DLL which does not contain those functions can be installed in the 
windows directory by some applications.
