emacs works differently than MSVC workbench editor does.

If you want to make sure a block of code is in its   
"correct-tab-position,"
try the following:
0. use ctrl-space to mark the beginning of the block
1. move the cursor to the end of the block by any means
2. go meta-ctrl-\ (backslash)

I m 100% sure that will work in almost all versions of g/x/emacs.
MSVC workbench? Not sure though.
