You can use CDaoRecordset::MoveFirst to move to the first record in your
recordset and next use CDaoRecordset::FindFirst ( LPCTSTR lpszFilter ) where
lpszFilter is string expression (like the WHERE clause in an SQL statement
without the word WHERE) used to locate the record.

Don't forget to test if you find more that one record in your recordset,
that satisfy 'lpszFilter'.

Hope this help you !
