-----From: Gerry Sweeney 


Carlos,

Try a '%'(percent) sign. For example:-

"SELECT * FROM mytable WHERE name LIKE A%"

Will return all records where the name field begins with A

or

"SELECT * FROM mytable WHERE name LIKE %A%"

Will return all records where the name field contains A

Hope this helps.
