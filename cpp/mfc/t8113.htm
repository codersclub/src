Hi Adam,

U just force the compiler to use one calling convention throughout U'r program.
So do include extern "C" above the function prototype like..

extern "C"
fn() ; // may be in the header file if 'fn()' is in a dll

extern "C"
fn()
{
 // fn code here...
}
