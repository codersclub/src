See article Q132960 in the Microsoft Developer Knowledgebase 
(http://www.microsoft.com/kb).  I searched on "mfcdatabase foxpro" and hit 
the article.   You just need to change the fourth argument of the RFX_Text 
call.

-----From: "Frederic Steppe" 

You have to modify the call to RFX_Text, which defaults to 255 chars max size. 
 Just code the fourth parameter (nMaxLength) to 254 or less.
