I think you really mean "IMsocDocument".

If that's the case, then: no, not in Version 4.0 or 4.1. In MFC 4.1, though, 
you'll find a sample which shows you how to add binder support to your 
COleServerDoc-derived document. Maybe some later version of MFC will have 
intrinsic support for those classes.

If you already have the Microsoft Office COmpatibility Developer's Kit (or 
whatever the heck those guys decided to call it), though, you should have a 
version of this same sample. For now, I'm told that the MSOCDK is the only 
way that I think you're supposed to know about this interface, so I guess you 
already have it.
