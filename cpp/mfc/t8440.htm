>     If so, can I use classes like CMDIChildWnd and other parts of MFC?
>     I can pass handles to windows from Powerbuilder to the DLL if that's any he=

  You must pass the HWND of your PB's window, and create a MFC Windows
as its child. I'v just do the same thing in my project, more than PB, I
create it on VB,Delphi... They all 
work ok. I don't think it is difficult, so just try it.
  BTW: I didn't create a MDI Child Window in others MDI Client. I just
create the MDI Child in the other app, then create a MFC windows as
child of MDI Child window.
  Hope it will be helpful.
