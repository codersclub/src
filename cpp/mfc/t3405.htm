My partner just came back from MacWorld where he talked to the CodeWorks
people directly. Here's my understanding of what they offer:

Code Warrior for Windows can compile MFC but NOT PowerPlant (CodeWork's
Mac-based class library)
Code Warrior for Mac can compile PowerPlant but NOT MFC

Neat huh?

Basically, MS hasn't released a x-Platform version of MFC to other tool
developers, you can only get it with Dev Studio. And Code Works has no plans
to port PowerPlant to Windows.

In summary: Code Warrior doesn't solve any x-Platform issues if you plan to
use either MFC or PowerPlant.

On a similar note: does anyone have any experience x-compiling from MSVC++
and MFC to the Macintosh (either PPC or 68k)? I'd love to hear about the
issues involved.
