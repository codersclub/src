	 WS_CHILD.

     If you don't have/can't access the mfc-l archive...
     
     What I did to solve this problem was to add the CONTROLPARENT style to 
     the nested sheet.  In OnInitDialog() of the inner 
     CPropertySheet-derived object, add that style.  Here's what I did:
     
     BOOL CHypoPortfolioSheet::OnInitDialog(void) {
        ModifyStyleEx(0,WS_EX_CONTROLPARENT);
        return CPropertySheet::OnInitDialog();
     }
