Question: Have you installed the beta 2 release of Microsoft Internet Explorer 
3.0? If so, that may be the cause of the trouble. The beta 2 release included 
a COMCTL32.DLL that had some bugs in the toolbar code, for which the symptoms 
are similar to what you've described.

The next Internet Explorer release (due sometime later this month) will have a 
fixed COMCTL32.DLL. If you can't wait that long, you should try to install an 
older version of it (e.g., from another Win95 machine).

John ("not speaking for Microsoft") Elsbree
