You can skip checking all the attributes of the property page by using the 
"Resource..." command in the "Insert" menu.  When the "Resource" dialog comes 
up, expand the tree control underneath "Dialogs"--it'll show a couple stock 
sizes for property pages and if you insert one of them, the new resource will 
be set up the way it should.
