Hi,  
See the documentation and code for CWnd::OnToolHitTest().  The default is
to center the tooltip for a non-button control.  You can override this
function and remove the TTF_CENTERTIP flag from the TOOLINFO structure.

Best regards,
--dan

This flag is set by default if the control is not a button.
