     Here are the options I had to use in VC++ 1.52c to get this to work.  
     I don't use 4.2 (yet...don't ask), but these may help you out:
     
     The options I need to have set for Debug or Release to get a .LIB to 
     link are:
     
     /Gs        -       Code Generation | Disable Stack Checking
     /Gx-       -       Memory Model | Assume 'extern' and Uninitialized    
                                        Data 'far'
     /GA        -       Windows Prolog/Epilog | Protected Mode Application  
                                                Functions
