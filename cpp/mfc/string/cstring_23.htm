CString::MakeReverse

Этот метод полностью изменяет порядок символов в этом объекте CString, т.е. реверсирует строку.

void MakeReverse( );

Пример

Следующий пример демонстрирует использование CString::MakeReverse.

// пример для CString::MakeReverse

CString s( "abc" );

s.MakeReverse();

ASSERT( s == "cba" );
