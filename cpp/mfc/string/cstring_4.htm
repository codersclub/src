CString::Compare

Этот метод сравнивает объект CString с другой строкой, используя Windows CE функцию lstrcmp .

int Compare(

LPCTSTR lpsz )

const;

Параметры

lpsz

Определяет другую строку, используемую для сравнения.

Возвращаемое значение

Нуль, если строки идентичны, меньше 0, если этот объект CString - меньше чем lpsz, или больше 0 если этот объект CString, больше чем lpsz.

Пример

Следующий пример демонстрирует, что использование CString::Compare.

// пример для CString::Compare

CString s1( "abc" );

CString s2( "abd" );

ASSERT( s1.Compare( s2 ) == -1 ); //Сравниваются с другим CString.

ASSERT( s1.Compare( _T("abe" ) ) == -1 ); // Сравниваются с LPTSTR строкой.
