CString::GetLength

Этот метод возвращает количество символов в данном объекте CString. Это количество не включает null-терминатр.

int GetLength () const;

Возвращаемое значение

Количество байтов в строке.

Пример

Следующий пример демонстрирует использование CString::GetLength.

// пример для CString::GetLength

CString s ("abcdef");

ASSERT (s.GetLength () == 6);
