CString::Empty

Этот метод делает объект CString пустой строкой и освобождает соответствующую память .

void Empty( );

Пример

Следующий пример демонстрирует использование CString::Empty.

// пример для CString::Empty

CString s ("abc");

s. Empty ();

ASSERT (s. GetLength () == 0);
