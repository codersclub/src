CString::FindOneOf

Этот метод ищет в строке первый символ, который соответствует любому символу, содержавшемуся вlpszCharSet.

int FindOneOf(
LPCTSTR lpszCharSet )
const;

Параметры

lpszCharSet

Определяет строку содержащие символы для соответствия.

Возвращаемое значение

Отсчитываемый от нуля индекс первого символа в этой строке, который находится также вlpszCharSet; или -1, если нет никакого соответствия.

Пример

Следующий пример демонстрирует использованиеCString::FindOneOf.

// пример для CString::FindOneOf
CString s( "abcdef" );
ASSERT( s.FindOneOf( _T( "xd" ) ) == 3 ); // 'd' первое соответствие
