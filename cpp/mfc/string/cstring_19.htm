CString::Left

Этот метод извлекает первые (крайние левые) nCount символов из объекта CString и получает копию извлеченной подстроки. Если nCount превышает строковую длину, извлекается вся строка. Left подобен функции Бейсика LEFT$, за исключением того, что индексы являются отсчитываемыми от нуля.

CString Left(

int nCount)

const;

Параметры

nCount

Определяет число символов, чтобы извлечь из этого объекта CString.

Возвращаемое значение

Объект CString, содержащий копию указанного диапазона символов. Обратите внимание, что возвращенный объект CString может быть пуст.

Пример

Следующий пример демонстрирует использование CString::Left.

// пример для CString:: Left

CString s( _T("abcdef") );

ASSERT( s.Left(2) == _T("ab") );
