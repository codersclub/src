CString::ReverseFind

Этот метод ищет в объекте CString последнее соответствие подстроке. Метод подобен _tcsrchr run-time функции.

int ReverseFind(

TCHAR ch )

const;

Параметры

ch

Определяет символ, чтобы искать.

Возвращаемое значение

Возвращает индекс последнего символа в этом объекте CString, который соответствует требуемому символу; Или -1, если символ не найден.

Пример

// Пример для CString:: ReverseFind

CString s( "abcabc" );

ASSERT( s.ReverseFind( 'b' ) == 4 );
