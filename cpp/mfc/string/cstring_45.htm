CString::operator[]

Этот перегруженный оператор [] возвращает отдельный символ, указанный отсчитываемым от нуля индексом в nIndex. Этот оператор - удобная замена GetAt метода.

Вы можете использовать оператор [] , чтобы получить значение символа в CString, но Вы не можете использовать его, чтобы изменить значение символа в CString.

TCHAR operator [](

int nIndex )

const;

Параметры

nIndex

Определяет отсчитываемый от нуля индекс символа в строке.

Пример

Следующий пример демонстрирует использование CString::operator[ ].

// пример для CString::operator [ ]

CString s ("abc");

ASSERT( s[1] == 'b' );
