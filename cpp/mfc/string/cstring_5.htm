CString::CompareNoCase

Этот метод сравнивает объект CString с другой строкой, используя функцию lstrcmpi. Т.е. игнорируя регистр.

чтобы получить правильное специфическое для языка упорядочениеТекущий язык используется, .

int CompareNoCase(

LPCTSTR lpsz)

const;

Параметры

lpsz

Определяет другую строку, используемую для сравнения.

Возвращаемое значение

Нуль, если строки идентичны; меньше 0, если этот объект CString меньше чем lpsz , или больше 0 если этот объект CString, больше чем lpsz. Регистр игнорируется

Пример

Следующий пример демонстрирует использование CString::CompareNoCase.

// пример для CString:: CompareNoCase

CString s1( "abc" );

CString s2( "ABD" );

ASSERT( s1.CompareNoCase( s2 ) == -1 ); // Сравниваются с CString.

ASSERT( s1.Compare( _T("ABE" ) ) == -1 ); // Сравниваются с LPTSTR строкой.
