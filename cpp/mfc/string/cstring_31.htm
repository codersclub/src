CString::SetAt

Этот метод заносит отдельный символ по указанному индексу.SetAt не будет увеличивать строку, если индекс превышает границы существующей строки.

void SetAt (

int nIndex,

TCHAR ch);

Параметры

nIndex

Определяет отсчитываемый от нуля индекс символа в объекте CString. nIndex параметр должен быть больше 0или равняться 0 и меньше чем значение, возвращенное GetLength. Отладочная версия Microsoft Foundation Class Library проверяет правильность границ nIndex; Release версия этого не делает.

ch

Определяет символ для вставки.
