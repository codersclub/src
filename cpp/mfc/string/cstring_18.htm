CString::IsEmpty

Этот метод определяет, пуст ли объект CString.

BOOL IsEmpty() const;

Возвращаемое значение

Отличо от нуля, если объект CString имеет нулевую длину; иначе, возвращается 0.

Пример

Следующий пример демонстрирует использованиеCString:: IsEmpty.

// пример для CString:: IsEmpty

CString s;

ASSERT (s. IsEmpty ());
