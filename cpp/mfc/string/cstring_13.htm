CString::GetAt

Этот метод возвращает символ с указанным индеком (номером символа в строке). Оператор [] - удобный заменитель для GetAt.

TCHAR GetAt (

int nIndex)

const;

Параметры

nIndex

Определяет индекс символа в объекте CString считая от нуля . Параметр nIndex должен быть больше 0 или равняться 0 и меньше чем значение, возвращенное GetLength. Отладочная версия Microsoft Foundation Class Library для Windows CE проверяет правильность границ nIndex; Release версия этого не делает.

Возвращаемое значение

TCHAR, содержащий символ указанной позиции строки. Отдельный символ находящийся по отсчитываемому от нуля индексу.

Пример

Следующий пример демонстрирует использование CString::GetAt.

// пример для CString::GetAt

CString s( "abcdef" );

ASSERT( s.GetAt(2) == 'c' );
