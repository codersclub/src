CString::LoadString

Этот метод читает строку-ресурс Windows CE с идентификатором nID в существующий объект CString.

BOOL LoadString (

UINTnID );

Параметры

nID

Определяет идентификатор строки-ресурса Windows CE.

Возвращаемое значение

Отличное от нуля, если загрузка ресурса была успешна; иначе - нуль.

Пример

Следующий пример демонстрирует использование CString::LoadString.

// пример для CString::LoadString

#define IDS_FILENOTFOUND 1

CString s;

if (! s.LoadString( IDS_FILENOTFOUND ))

{

AfxMessageBox("Error Loading String: IDS_FILENOTFOUND");

...

}
