CString::CString

Эти конструкторы инициализируют новый объект CString с указанными данными.Поскольку конструкторы копируют входные данные в новую область памяти, Вы должны знать, что может возникнуть исключительная ситуация переполнения памяти . Некоторые из этих конструкторов действуют как конверсионные функции. Это позволяет Вам заменять объект CString другими строками, например, использовать LPTSTR, когда ожидается CString в качестве параметра.

Каждый из этих конструкторов инициализирует новый объект CString с указанными данными.

Виды конструкторов

CString( ); 
CString(
  const CString& stringSrc ); 
CString(
  TCHAR ch, 
  int nRepeat = 1 );
  
CString(
  LPCSTR lpsz );
CString(
  LPCWSTR lpsz );
CString(
  LPCTSTR lpch,
 int nLength );
CString(
  const unsigned char* psz );

Параметры

stringSrc

Определяет существующий объект CString, который будет скопирован в этот CString объект.

ch

Определяет единственный символ, который будет повторен nRepeat количество раз.

nRepeat

Определяет количество повторений ch.

lpch

Определяет указатель на массив символов длины nLength, без учета символа конца строки.

nLength

Определяет количество символов в lpch.

psz

Определяет строку, завершаюшуюся нулем, которая будет скопирована в этот CString объект.

lpsz

Определяет строку, завершаюшуюся нулем, которая будет скопирована в этот CString объект.

Замечания

Некторые формы конструктора имеют специальные назначения:

·   CString ( LPCSTR lpsz) Создает Unicode CString из строки ANSI.

·   CString ( LPCTSTR lpsz) Вы можете использовать этот конструктор, чтобы загрузить строку как показано в примере ниже.

·   CString ( LPCWSTR lpsz) Создает CString из Unicode строки.

·  CString ( const unsigned char* psz) Позволяет Вам создаватьCString из указателя на unsigned char.

Пример

Следующий пример демонстрирует использование CString::CString.

// пример для CString:: CString

CString s1; // Пустая строка

CString s2 ("кот"); // Из C-строки

CString s3 = s2; // конструктор копии

CString s4 (s2 + " " + s3); // из строкового выражения

CString s5 ('x'); // s5 = "x"

CString s6 ('x', 6); // s6 = "xxxxxx"

CString s7 ((LPCTSTR) ID_FILE_NEW); // s7 = "Создать новый документ"

CString city = "Филадельфия"; // НЕ оператор присваивания
