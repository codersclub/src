CString::operator <<, >>

Оператор << для формирования дампа диагностики и сохранения в архив.

Используйте оператор извлечения (>>) для того, чтобы загрузить из архива.

CDumpContext операторы допустимы только в Отладочной версии MFC для Windows CE.

friend CArchive& AFXAPI operator <<(

CArchive& ar,

const CString& string );

friend CArchive& AFXAPI operator >>(

CArchive& ar,

CString& string );

friend CDumpContext& AFXAPI operator <<(

CDumpContext& dc,

const CString& string);

Параметры

string

Определяет объект CString для записи.

Пример

Следующий пример демонстрирует использование CString::operator <<, >>.

// пример для CString::operator <<, >>

extern CArchive ar;

CString s( "abc" );

*ifdef _DEBUG

afxDump <<s; // Выводит значение (abc)

afxDump <<*s; // Выводит адрес

*endif

if( ar.IsLoading() )

ar >> s;

else

ar << s;
