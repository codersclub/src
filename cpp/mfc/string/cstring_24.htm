CString::MakeUpper

Этот метод переводит символы объекта CString к символам в вверхнем регистре.

void MakeUpper();

Пример

Следующий пример демонстрирует использование CString::MakeUpper.

// пример для CString::MakeUpper

CString s( "abc" );

s.MakeUpper();

ASSERT( s == "ABC" );
