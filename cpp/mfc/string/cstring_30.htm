CString::Right

Этот метод извлекает последние nCount символов этого объекта CString и возвращает копию извлеченной подстроки. Если nCount превышает строковую длину, то извлекается вся строка . Right подобна функции Бейсика RIGHT$, за исключением того, что индексы являются отсчитываемыми от нуля.

CString Right(

int nCount )

const;

Параметры

nCount

Определяет число символов, чтобы извлечь из этого объекта CString.

Возвращаемое значение

Объект CString, который содержит копию указанного диапазона символов. Обратите внимание, что возвращенный объект CString может быть пуст.

Пример

Следующий пример демонстрирует использование CString::Right.

// пример для CString::Right

CString s( _T("abcdef") );

ASSERT( s.Right(2) == _T("ef") );
