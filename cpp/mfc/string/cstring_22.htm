CString::MakeLower

Этот метод переводит символы объекта CString к символам в нижнем регистре.

void MakeLower();

Пример

Следующий пример демонстрирует использование CString::MakeLower.

// пример для CString::MakeLower

CString s ("ABC");

s. MakeLower ();

ASSERT (s == "abc");
