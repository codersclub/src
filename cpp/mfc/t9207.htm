Hi,

I am doing exactly similar thing, initially the OLE control was not visible in
resource editor in test mode. Later I opened the resource file and moved
OLE control lines to the starting of the dialog's resource code, before
any other control is added to dialog. Program worked fine and the control is
visible in the test mode also. 

I have no idea why this happenes, but it worked. Hope this helps.
