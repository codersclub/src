There is a comment on this in one .vbx suppliers manual.  In part, it 
reads:

messages sometimes do not appear in calss wizard for vbx controls. If 
the VBX custom controls were not explicitly loaded in App Studio using 
Install Controls, the messages will not show up.  To rectify this 
problem, click on App Studios File menu, select Install controls to 
install the controls, and delete each supposedly "installed control".  
Immediately re-install the controls by clicking on the vbx file name and 
then clicking install.  The message mapping will then appear in Class 
Wizard.

Hope this helps,

Lee Thompson
