Hi Boerge,

Try calling this in the OnInitialUpdate of your CRichEditView:

void CYourRichEditView::OnInitialUpdate()
{
 CRichEditView::OnInitialUpdate();

 /* SET THE WORDWRAPPING */
 m_nWordWrap = WrapNone; // No automatic word wrapping
}

Hope this helps.
