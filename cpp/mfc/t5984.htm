Enviroment: VC++ 4.0 and Windows NT 3.51

I'm having trouble getting the help button to work
on property sheet/pages.  

Calling code looks like:

DaClass::DaMethod()
{
        CSheet cSheet ("Caption");
	CPage *pPage = (CPage *) cSheet.GetPage (0);

        // turn on the help button.  This
        // gets it to show up, but its *grayed out*.

	cSheet.m_psh.dwFlags |= PSH_HASHELP;
	cPage->m_psp.dwFlags |= PSH_HASHELP;

        if (cSheet.DoModal() == IDOK)
        {
          ...
        }
}

Online help says that a WM_NOTIFY carrying a PSN_HELP 
will be sent to my page when then help button is pressed.

How do I intercept this message?  

A assumed the problem was that I didn;t have a handler, so I tried 
to put a WM_NOTIFY manually in my page MESSAGE_MAP but got a syntax error
indicating I was missing some params.  A WM_NOTIFY specific
to PSN_HELP is *not* provided by Class Wizard (for me atleast).

How do I get the property sheet help button to activate?
