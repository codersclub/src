I hope you are not creating the ( CMiniFrameWnd derived ) window 
on stack.

Learn from others mistakes!!
(If thats any help!)
Sanjay

[Moderator's note: The most common mistake people make in MFC is to
create something on the stack and not realize that it will get deleted
as soon as the routine returns.  It drives people from the X Window
world crazy, and will probably do the same for Java folks, neither of
whom have to worry about this in many cases.]
