[Moderator's note: For those of you wondering what a pretty printer is,
it's a program that formats source code so that it looks nice on a
printer.  It can put things in different typefaces (bold for routine
names, proportional font for comments, etc.), handle pagination, and
create an index.  They aren't as popular as they were when we all used
text terminals.]

I used to use an old version FinePrnt.exe (from I don't know where), but   
it doesn't accept long filenames.

I now use a Word macro which turns comments into italics, etc. adds a   
header and prints it out with line-numbers down the side.

It's pretty straight forward, but e-mail me if you want a copy.
