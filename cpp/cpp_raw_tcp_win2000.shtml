Библиотека Raw TCP/IP для Windows 2000

Автор: Barak Weichselbaum.

Эта библиотека позволяем Вам получить больший контроль над протоколом TCP/IP, а именно, создавать свои IP заголовки в пакетах, указывать по своему усмотрению IP-адреса отправителя (!) и получателя, TTL и номера портов.

Библиотека содержит главный класс CSpoofSocket, который использует WinSock 2 API. В настоящий момент библиотека поддерживает только отправку пакетов (приёма нет). Так же присутствуют два класса CTCPSocket и CUDPSocket, которые позволяют полноценно работать с UDP/TCP пакетами.


Платформа: Windows 2000, Visual C++ 6.0
