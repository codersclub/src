Увеличительное стекло

Автор: Alex Farber.

Компилятор: C++ 6.0, NT 4.0 или выше, Win95 или выше

Увеличительное стекло - это приложение, которое помещает иконку в system tray. Увеличение активируется кликом по иконке. После этого при клике левой кнопкой мыши в любом месте экрана появится увеличительное стекло.

Использованные материалы:

  • Проект CamStudio. Использование невидимого окна для перехвата событий мышки со всего экрана. http://www.atomixbuttons.com/vsc
  • Chris Maunder. Добавление иконок в System Tray. http://www.codeproject.com/shell/systemtray.asp
  • Chris Maunder. Создание приложения, не имеющего иконки на панели задач. http://www.codeproject.com/docview/notaskbaricon.asp
  • Jeffrey Richter. Программирование приложений для Microsoft Windows. Microsoft Press 1999. Приложение B. Взлом сообщения.
