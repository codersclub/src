FDDCopy Версия 1.0

Авторы: Максим Киселёв и Ахметгалиев Роман

Компилятор: Visual C++

FDDCopy создаёт точную копию исходного диска и записывает её на результатирующий диск. Программа демонстрирует пример как считывать/записывать сектора при помощи DeviceIoControl под Windows 95/98/Millenium, под Window 2k/NT4.0 это делается через CreateFile/ReadFile/WriteFile.

ЛИЦЕНЗИЯ

Программа распростаняется свободно, без ограничения права копирования. Программа протестирована нами, по мере возможности.
