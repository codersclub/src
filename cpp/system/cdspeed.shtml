CDSpeed Версия 1.0

Автор: Максим Киселёв

Компилятор: Visual C++

CDSpeed это программа для управления скоростью CD-ROM приводов. Точнее, для ограничения максимальной скорости привода с целью уменьшить шум, когда не требуется читать данные очень быстро (например, при прослушивании MP3 музыки или в играх). Заодно улучшает читаемость и предотвращает разрыв некачественных дисков в скоростных приводах.

Запуск программы под Windows NT/2000

Я планирую включить встроенную поддержку Windows NT/2000 в одну из следующих версий программы. Но если очень нужно, можно запустить текущую версию программы под этими ОС - для этого нужно установить драйвер поддержки ASPI.

Если у Вас в компьютере установлен SCSI адаптер фирмы Adaptec - Вам крупно повезло. Скачайте с сайта Adaptec файл ASPI32.EXE (URL: http://www.adaptec.com/worldwide/support/driverdetail.html?cat=%2fOperating+System%2fMicrosoft+Windows+NT&filekey=aspi32.exe), если нет желания заполнять анкету, то вот ссылка прямо на файл: ftp://ftp.adaptec.com/software_pc/aspi/aspi32.exe. Установите это драйвер и все, программа должна работать.
