Полный список функции GINA

Функция Описание
WlxActivateUserShell Активирует пользовательский шелл
WlxDisplayLockedNotice Позволяет GINA DLL отображать информации о заблокированности
WlxDisplaySASNotice Winlogon вызывает эту функцию, когда пользователь авторизуется в системе
WlxDisplayStatusMessage Winlogon вызывает эту функцию, когда отображается сообщение о состоянии
WlxGetConsoleSwitchCredentials Winlogon вызывает эту функцию для чтения прав пользователя, чтобы передать их в текущую сессию
WlxGetStatusMessage Winlogon вызывает эту функцию, чтобы получить сообщение о состоянии
WlxInitialize Инициализация GINA DLL
WlxIsLockOk Проверка заблокированности рабочей станции
WlxIslogoffOk Проверка на logoff
WlxLoggedOnSAS Winlogon вызывает эту функцию, когда получает сообщение о нажатии Ctrl-Alt-Del, если рабочая станция не заблокирована и пользователь залогинен в системе
WlxLoggedOutSAS Winlogon вызывает эту функцию, когда получает сообщение о нажатии Ctrl-Alt-Del, и если в системе нет ни одного залогиненного пользователя
WlxLogoff Уведомляет GINA DLL, что была попытка выйти из системы
WlxNegotiate Текущая версия Winlogon, который используется с GINA DLL
WlxNetworkProviderLoad Winlogon вызывает эту функцию после того, как загрузит сетевого провайтеда для сбора авторизационной и идентификационной информации
WlxRemoveStatusMessage Winlogon вызывает эту фкункцию для прекращения отображения сообщения о состоянии
WlxScreensaverNotify Позволяет GINA взаимодействовать со скринсейвером
WlxShutdown Winlogon вызывает эту функцию перед завершением работы, чтобы дать возможность GINA выполнить завершающие задачи
WlxStartApplication Winlogon вызывает эту функцию, когда системе необходимо запустить приложение в контексте пользователя
WlxWkstaLockedSAS Winlogon вызывает эту функцию, когда получает сообщение о нажатии Ctrl-Alt-Del, если рабочая станция заблокирована
