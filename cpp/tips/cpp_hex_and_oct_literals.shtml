Шестнадцатиричные и восьмиричные литералы

Перед шестнадцатиричными и восьмеричными литералами ставится специальный префикс. Шестнадцатиричный литерал предваряется префиксом '0x':

  int n =0x1000; // десятичное значение n равно 4096

А восьмеричный литерал - префиксом 0 (ноль):

  n = 01000; // десятичное значение n равно 512
