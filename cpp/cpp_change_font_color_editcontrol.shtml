Изменение параметров Edit Control.

Автор: Max Raskin

Пример демонстрирует, как изменить цвет текста, цвет бэкграунда, шрифт текста (включая, наклон, подчёркивание и жирность шрифта) в таком элементе управления как Edit. А самое главное - этот пример не использует MFC, как говорится, чистое API :)

Совместимость: Microsoft Visual C++
