Цифровые часики

Автор George Chastain.

Класс CGCClock позволяет программисту без особых затрат поместить цифровые часы в своё приложение. Этот класс наследован из CGCColorStatic класса, для обеспечения управления цветами фона и текста, используемыми шрифтами, типом бордюра и т.д.

Класс CGCClock позволяет вести отсчёт времени как в 12 часовом формате, так и в 24 часовом (иногда называемом военным форматом).

Как использовать CGECClock

  • 1. Добавьте класс CGCClock в Ваш проект.
  • 2. Добавьте класс CGCColorStatic в Ваш проект.
  • 3. Поместите на форму элемент "статический текст", используя редактор ресурсов.
  • 4. Можно установить стиль "Align Text" в "Center" и поставить стиль "Center vertically".
  • 5. Используйте ClassWizard для создания переменной члена типа CGECClock.
  • 6. Вызовите метод Start() для запуска часов.
