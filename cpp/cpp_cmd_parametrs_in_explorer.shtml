Задаём параметры командной строки в Проводнике.

Автор: Nick Carruthers

Скачать исходник - 376 Kb

Описание

Пример показывает как заставить Ваш Проводник задавать параметры командной строки при запуске любого приложения. Достаточно нажать правой кнопкой мыши на исполняемом файле и выбрать в появившемся меню "Выполнить с параметрами". Несколько откомпилированных DLL прилагаются. (прим. Вебмастера: на WinNT заработала третья по счёту DLL).

Компилятор: Visual C++ 6.0

Откомпилированные версии

В Архиве содержится несколько версий откомпилированного примера:

  • bin\ReleaseUMinDependency\ - Unicode
  • bin\DebugU\ - Unicode
  • bin\ReleaseMinDependency\
  • bin\ReleaseMinSize\
  • bin\Debug\
  • bin\ReleaseUMinSize\ - Unicode

Чтобы зарегистрировать нужную версию DLL (не забудьте, что Unicode версии не будут правильно работать в Win95/98), скопируйте файл в системную директорию (Windows System Directory), и запустите из командной строки "RegSvr32 CmdLineExt.dll". После этого достаточно в Проводнике правой кнопкой мыши щёлкнуть на EXE-шник и выбрать в появившемся меню "Run with Parameters".
