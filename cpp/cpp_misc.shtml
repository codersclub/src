INF2HTML V0.92 - Конвертор файлов помощи от OS/2 в HTML формат 15.09.2002г.
INF2HTML - это небольшая консольная утилита для Win32, которая преобразует стандартные файлы справки системы OS/2 (.INF или .HLP) в набор HTML файлов, причем кажая отдельная страница справки (панель) преобразуется в отдельную HTML страницу. INF2HTML автоматически создает ссылки между страницами, что позволяет легко просматривать полученный HTML код с помощью любого HTML браузера.
Компилятор: Borland C++ 5, Visual C++ 6

Увеличительное стекло - это приложение, которое помещает иконку в system tray. Увеличение активируется кликом по иконке. После этого при клике левой кнопкой мыши в любом месте экрана появится увеличительное стекло.

Использование директивы #import в Visual C++ 19.07.2000г.
Директива #import введена в Visual C++, начиная с версии 5.0. Её основное назначение облегчить подключение и использование интерфейсов COM, описание которых реализовано в библиотеках типов. В данной статье я попытаюсь объяснить то, как работает эта директива и привести несколько примеров её использования. Надеюсь, после этого вы тоже найдёте её полезной.
