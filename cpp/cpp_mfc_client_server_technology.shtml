Программирование технологии клиент-сервер с использованием Windows Sockets и MFC.

Автор: Сергей Дмитриев

Сервер обеспечивает соединение с несколькими клиентами. Посылает и принимает от них текстовые сообщения.

Компилятор: Visual C++

Окно сервера:

Послать сообщение - либо double click на имени клиента в списке, либо выбрать его и нажать Send. Послать текст всем клиентам - Send All.

Окно первого клиента:

Настройка клиента: имя(отображается на сервере) и адрес сервера, к которому подсоединяемся, указывается в отдельном файле. Имя этого файла можно передать как параметр командной строки, в противном случае параметры устанавливаются по умолчанию.
