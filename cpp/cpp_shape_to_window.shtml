Рисуем собственное окно.

Автор: Brad Zasada

Пример показывает как можно преобразить вашу программу, используя вместо стандартного окна Windows своё собственное, нарисованное от руки :). Оказывается в API функция сделать это не составляет труда, судя по маленькому размеру исходника.
Все ресурсы находятся в файле "main.res", включая картинки.
Проект собран и откомпилирован мною на MS Visual C++ 4.2.
