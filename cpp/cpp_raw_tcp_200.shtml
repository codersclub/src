Библиотека Raw TCP/IP для Windows 2000

Автор: Barak Weichselbaum.

Это обновление данной библиотеки, расширяющей возможности протокола TCP/IP, и позволяющей пользователям создавать собственные TCP/IP заголовки, которые позволяют организовать IP spoofing и другие виды атак.

Новые возможности во второй версии включают в себя: Полный контроль над сокетом (асинхронный и блокирующий), который может полностью заменить CAsyncSocket, полный контроль над IP и TCP заголовками.

Как обычно головной класс : CSpoofSocket, который работает через socket2 API, изменяя некоторые заголовки, такие как: Source IP (особенно для спуфинга), TTL, тип сервиса.

Также присутствуют классы CTCPSocket,CUDPSocket, CICMPSocket которые наследуются от  CSpoofSocket, и Вы можете использовать их для посылки/приёма UDP/TCP/ICMP пакетов.

Примеры поставляемые с библиотекой: ping, traceroute, простейшие атаки, простой TCP сервер.


Платформа: Windows 2000, Visual C++ 6.0 (3 примера без использования MFC)
