Программа отправки пейджинговыйх сообщений через e-mail.

Автор: Boris M. Chomitch

Dialog-Based MFC aplication. Позволяет отправлять пейджинговые сообщения абонентам Ростовксой пейджинговой компании ААА через e-mail. Для этого отправляется письмо с текстом сообщения на виртуальный почтовый ящик: <Номер пейджера>@pager.aaanet.ru Данный метод отправки сообщений через e-mail является своеобразным стандартом, поэтому программа может быть переделана под любого пейджингового оператора.

Для хранения настроек используется реестр Windows. Запускается только один экземпляр приложения. Окно приложения умеет сворачиваться в SystemTray. Использовались классы CSMTP и CMailMessage с www.codeguru.com

Компилятор: Visual C++
